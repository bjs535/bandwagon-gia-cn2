**Q：搬瓦工 CN2 GIA 和 CN2 GT 有什么区别？**

GIA 是全程走 CN2 专属线路，GT 只是部分路由经过 CN2，普通段仍是 163 骨干网。晚高峰时 GT 的表现明显不如 GIA，如果预算允许，直接选 GIA。
